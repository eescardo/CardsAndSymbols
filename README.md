CardsAndSymbols
----------------

Generates sets of cards using projective plane geometry, where any card pair has exactly one symbol in common (like game Spot it!)

## Prerequisites

### Install .NET SDK 10.0 on macOS

1. **Using Homebrew (recommended):**
   ```bash
   brew install --cask dotnet-sdk
   ```

2. **Or download from Microsoft:**
   Visit https://dotnet.microsoft.com/download/dotnet/10.0
   Download and install the macOS SDK

## Building and Running

### Build the solution:
```bash
dotnet build CardsAndSymbols.sln
```

### Run the application:
```bash
cd CardsAndSymbols
dotnet run
```

### Run the console test:
```bash
cd ConsoleTest
dotnet run
```
